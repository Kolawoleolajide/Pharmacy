## ğŸ“Œ Project Overview

This project analyzes sales and profitability data for a European pharmacy chain operating across multiple countries, regions, and pharmacies. Using transaction-level sales data, the goal was to uncover how performance varies across geography, products, and promotions, and to provide decision-ready insights for business stakeholders.

The dashboard focuses on **revenue, units sold, margin, and margin percentage**, with drill-down capabilities that allow users to move seamlessly from high-level summaries to detailed pharmacy and product views.

## ğŸ” Key Insights

* Revenue and profitability are **geographically concentrated**, creating dependency on a small number of regions.
